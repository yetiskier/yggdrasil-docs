# Yggdrasil

Yggdrasil is a commercial geophysical processing suite by **Aesir Consulting
LLC**: one desktop application, with independently licensed processing
packages ("realms", each named from Norse mythology) that share
**one project and one 3D scene**. Buy the method you need today — every data
type you add later lands in the same project, on the same terrain, in the
same view.

![Inverted resistivity sections and GPR radar curtains rendered together beneath a translucent terrain and aerial-photo basemap](assets/img/centerville_scene_full.png)

*Inverted resistivity sections and migrated GPR radar profiles rendered
